## How to Test Sharing

1. Login as Alice
2. Create a document and add content
3. Click "Share" → select Bob → set permission to "view" → click Share
4. Open an incognito window → login as Bob
5. Bob sees the document under "Shared with Me"
6. Bob can view but cannot edit (view-only banner displayed)
